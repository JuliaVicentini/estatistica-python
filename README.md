# 📊 Estatística com Python — Alura

Notebooks desenvolvidos durante o curso de Estatística da [Alura](https://www.alura.com.br/), com foco na aplicação de conceitos estatísticos usando Python e pandas.

## 📚 Conteúdo

O curso utiliza dados reais da **Pesquisa Nacional por Amostra de Domicílios (PNAD 2015)** do IBGE, explorando variáveis como renda, idade, altura, sexo, cor e anos de estudo da população brasileira.

### Tópicos abordados

- **Tipos de variáveis** — qualitativas (nominais e ordinais) e quantitativas (discretas e contínuas)
- **Distribuição de frequências** — para variáveis qualitativas e quantitativas, com classes personalizadas e pela regra de Sturges
- **Histogramas e visualizações** — com `matplotlib` e `seaborn`
- **Medidas de tendência central** — média, mediana e moda
- **Medidas separatrizes** — quartis, decis, percentis e box-plot
- **Medidas de dispersão** — desvio médio absoluto, variância e desvio padrão
